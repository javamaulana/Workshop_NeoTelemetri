Pengenalan Git dan Github

Git adalah sistem kontrol versi yang digunakan untuk melacak perubahan dalam kode sumber selama pengembangan perangkat lunak. Git memungkinkan banyak pengembang untuk bekerja pada proyek yang sama tanpa mengganggu satu sama lain.

Fungsi Git

  • Melacak perubahan dalam kode sumber
  • Mendukung kolaborasi antara banyak pengembang
  • Menyimpan riwayat versi proyek
  • Memudahkan pengembalian ke versi sebelumnya
